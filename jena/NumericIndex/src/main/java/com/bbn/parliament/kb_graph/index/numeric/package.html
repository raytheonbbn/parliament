Provides classes and interfaces for creating an index that handles numeric data. The {@link com.bbn.parliament.kb_graph.index.numeric.NumericIndex} indexes a specific predicate and number type using the Berkeley DB Java Edition. The NumericIndex and {@link com.bbn.parliament.kb_graph.index.numeric.NumericIndexFactory} classes are abstract. Concrete implementations are provided as static nested classes. For instance, {@link com.bbn.parliament.kb_graph.index.numeric.NumericIndex.DoubleIndex} and the related {@link com.bbn.parliament.kb_graph.index.numeric.NumericIndexFactory.DoubleIndexFactory} provide an index and factory for {@link java.lang.Double}s.

The {@link com.bbn.parliament.kb_graph.index.numeric.NumericIndexPatternQuerier} claims triples of the form ?s <predicate-to-index> ?o. The index is intended to be accessed only for range queries.
